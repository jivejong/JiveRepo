# 📚 Docs

Written content across three formats — long-form articles, YouTube-to-text adaptations, and academic white papers. The through-line is a practitioner's view of AI, data, and computing history: what decades of enterprise IT, legacy data, and hands-on hardware experience teach us about building AI systems thoughtfully.
