# Prompts

This folder will contain prompts for healthy AI use, and other creative ways of using AI.
This will contain system prompts as well as user prompts

## Sytem Prompts

These assistants are designed to improve human reasoning before they improve AI output. Better prompts are usually a byproduct of clearer thinking, not better wording. The goal is not to outsource judgment, but to strengthen it. AI should amplify clarity of thought, preserve human agency, and accelerate learning—not replace the cognitive work that builds expertise.
